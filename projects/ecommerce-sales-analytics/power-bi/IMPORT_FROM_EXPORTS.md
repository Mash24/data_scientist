# Power BI — import from exports (fast path)

Aggregated CSVs are already in `exports/` (generated by `python scripts/run_pipeline.py`).

## Steps

1. Open **Power BI Desktop** → Get data → **Text/CSV**.
2. Import these files:

| File | Use on page |
|------|-------------|
| `exports/monthly_revenue.csv` | Executive — trend line, KPI cards |
| `exports/category_revenue.csv` | Categories — bar chart Top 15 |
| `exports/state_revenue.csv` | Executive — map/bar by state |
| `exports/payment_mix.csv` | Executive — donut |
| `exports/repeat_customers.csv` | Customers — KPI cards |
| `exports/reviews_vs_delivery.csv` | Customers — bar chart |

3. **No relationships needed** — each file is pre-aggregated for dashboard pages.
4. Add measures from `DAX_MEASURES.md` if you import raw tables later.
5. Apply theme color `#059669` to match analytics portfolio.
6. Publish → copy URL to `powerBiUrl` in portfolio metadata.
